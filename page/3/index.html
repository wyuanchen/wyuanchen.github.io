IP协议

2016-03-05
网络协议
IP

概述

  • IP 协议是一种不可靠、无连接的协议 。
  • 不可靠即不能保证 IP 数据报能够成功到达目的端，在传输过程中发生某种错误时，IP 协议会丢弃数据报，然后发送 ICMP 消息给信源端，其他任何要求的可靠性必须由上层来提供。
  • 无连接即 IP 协议并不维护任何关于后续数据报的状态信息，每个数据报都是相互独立的。因此，IP 数据报可以不按顺序接收。
